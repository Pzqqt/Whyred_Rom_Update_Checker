# Whyred_Rom_Update_Checker

A simple tool to check whyred's Rom update.

## Feature
- Write with Python3.
- Use sqlite3 to save data.
- Use your Telegram BOT to send you updated news.

## Required
- Python 3.7+

## Required Python Extension Packages
- beautifulSoup4
- requests
- sqlalchemy
- pyTelegramBotAPI
